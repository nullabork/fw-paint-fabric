# FW Paint v1.0.0

First release: a client-side Fabric mod for painting block gradients and noise fills, multiplayer-safe.

- **Gradient tool** — smooth colour/brightness blends along a marked line, with order modes
  (Colour, Brightness, Top % Dark/Light, B&W/Colour Diff), curve, chaos, deviation budget, and
  max-steps settings.
- **Noise tool** — flood-fill bounded by marker segments, driven by a seedable 3D noise field
  (Smooth / Perlin / Fractal) with per-axis scale and a live preview.
- **Pick mode** — fully manual block ordering by numbering rows.
- Modular block picker with must-use (✓) and exclude (✗) roles; palette from hotbar/inventory.
- Markers saved per world/dimension; in-world marker rendering.
- Build/release CI (tagged pushes publish a GitHub Release with the jar).

Targets: Minecraft **26.2** · Fabric Loader **0.19.3+** · Fabric API for 26.2 · Java **25**.
