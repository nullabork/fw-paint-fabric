# FW Paint v1.4.0

The preview release: both paint tools now show you exactly what they'll build — the noise tool as
a solid isometric cube sampled where you stand, the gradient as a cylinder run through the real
placement pipeline — and both can be blown up full screen.

Targets (unchanged): Minecraft **26.2** · Fabric Loader **0.19.3+** · Fabric API for 26.2 (Fabric
jar only) · Java **25**.

## Noise preview: a real cube, really placed

- The flat sample grid is gone: the preview is now a packed isometric cube whose three visible
  faces show the noise in 3D.
- It samples at **real world coordinates anchored at your feet** — the right face runs East, the
  left face South, the top face up — using exactly the maths placement uses (integer coordinates,
  same seed and scales). What you see is what painting that region would place.
- **Face-locked panning**: mouse-down on a face locks the drag to that face — the top face pans
  across the ground (X/Z), the side faces also pan up and down (drag up = coordinates up). Release
  to unlock. The anchor coordinates read out live under the cube.

## Gradient preview: the cylinder

- New preview under the gradient sliders: a hollow, open-topped cylinder — the top rim is the
  start of the gradient, the bottom rim the end.
- Every column of the cylinder runs the real placement pipeline — Curve, Chaos, per-column Step
  length wobble, and Variation stand-ins — so the column-to-column spread is the spread a real
  build would get. The preview is stable while you look and re-rolls when a setting changes.

## Expand (⛶)

- Both previews have a **⛶** button that opens them full screen over a dark backdrop with bigger
  sprites; **✗** or Esc returns to the settings. Handy when the settings column leaves the
  preview little room.

## Small stuff

- Noise Scale sliders now top out at 15 (was 64) — the range that's actually useful.
- Removed the duplicate Source button from the Gradient tab's right column.
- The in-game Help covers both previews.
