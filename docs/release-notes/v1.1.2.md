# FW Paint v1.1.2

Marker reliability release: end markers now genuinely stop Solid paint, the auto end marker scans
out of the face you click (with a live blue face + arrow preview), and a new **Help** tab puts a
drill-down player manual in-game.

Targets (unchanged): Minecraft **26.2** · Fabric Loader **0.19.3+** · Fabric API for 26.2 · Java **25**.

## End markers really stop Solid paint

- An end-marker cell now acts like a solid block for the Solid tool: an extruding column stops at
  it and a 3D fill never grows into it — even when the marker sits in air. Previously both relied
  on the marker happening to be a physical block, so air markers were sailed straight through.
- Multi-column extrude: re-clicking a face no longer pushes finished columns past their end
  markers. A column whose path already crossed its end marker stays finished; only the columns
  that haven't reached theirs continue (each still resuming from its own first air gap).
- Gradient and Noise placement were already safe — they only ever fill strictly between a
  start→end pair.

## Auto end marker, now face-driven

- The Settings toggle used to drop an end marker a full Marker-dist along your look axis, usually
  in mid-air. Now placing a start marker scans outward from **the face you clicked** — click a
  floor's top face and it scans straight up, a wall's side face and it scans out sideways — and
  marks the **first non-air block** it hits as the end. All air within Marker dist → the end lands
  at max distance.
- Dragging a line of start markers gives a matching line of ends: every start in the line scans
  along the face the drag began on.
- New in-world preview while marking (with the toggle on): the face you're aiming at is tinted
  translucent blue with a thin arrow along its normal showing where the scan will go. Hidden on
  blocks that are already start markers (clicking those removes instead), and shown down the whole
  line during a drag.
- A yellow hint under the Settings toggle sums it up.

## New: Help tab

- A fifth, last tab holding the in-game manual: every topic shows a one-line summary, clicking a
  title drills into detail paragraphs and nested sub-topics.
- Covers getting started + keybinds, markers (start/end/auto/marker dist), all three tools and
  their modes, every gradient mode and slider, the settings-preview vs placed-gradient
  distinction, and what a noise field actually is.
- Expansion and scroll position stick around while the game is running.
