# FW Paint v1.3.0

The colour release: a new **Finder** tab that ranks every block in the game by colour, and every
tool now measures colour the way the eye sees it (Oklab) — plus fixes that make sampled block
colours actually correct for the first time.

Targets (unchanged): Minecraft **26.2** · Fabric Loader **0.19.3+** · Fabric API for 26.2 (Fabric
jar only) · Java **25**.

## Finder tab

- Every placeable block in the game in one scrollable list, ranked by **Color** (a rainbow: grays
  first dark → light, then hue families with brightness snaking smoothly through each band) or
  **Brightness** (dark → light). Each row shows the block, its computed colour swatch, and its name.
- **Click any block** to select it — it highlights and auto-scrolls to the middle of the list, so
  its nearest colour neighbours sit just above and below. A small target button re-centres after
  you scroll away.
- Or pick a **pure colour**: a hue × saturation field plus a brightness slider. A coloured marker
  line shows exactly where that colour falls in the ordering.
- Colours come from each block's actual textures (resource packs included), computed once per
  session — browsing and re-sorting never recompute anything.

## Perceptual colour (Oklab) everywhere

- Gradient and Noise ordering, the brightness modes, Variation bands, and Solid's Closest
  colour/brightness matching now measure colour in **Oklab** — the perceptually uniform colour
  space — instead of luma + raw RGB. Blends space out the way the eye sees them, and "closest"
  actually looks closest.
- New Settings toggle: **Color match — Perceptual / Classic**. Perceptual is the default;
  Classic restores the exact pre-1.3.0 maths if you preferred the old picks.

## Colour sampling fixes

- **Red/blue swap**: sampled texture colours had their red and blue channels swapped (invisible
  while colours were only sorted, never shown). Brightness-based ordering was subtly wrong —
  reddish blocks read too dark, bluish too bright — and is now correct.
- **Tinted blocks**: grass, foliage, redstone, and other tint-coloured textures used to sample as
  near-white; their tint colour is now applied, so they sort as the colour you actually see.

## Small stuff
